# XAVIER — COMBAT ANTHEM

Xavier 캐릭터 전투 테마곡 랜딩페이지

## 배포 (GitHub Pages)

1. 이 레포지토리를 GitHub에 push
2. Settings → Pages → Source: `main` 브랜치, `/ (root)` 선택
3. 저장 후 `https://[username].github.io/[repo-name]/` 접속

## 파일 구조

```
index.html       ← 메인 페이지
media/
  main.mp4       ← 메인 테마: Polilla de ascuas
  track1.mp4     ← Track ver.1
  track2.mp4     ← Track ver.2
  track3.mp4     ← Track ver.3
  poster_*.jpg   ← 포스터 프레임
  cover.png      ← 앨범 커버
```
